---
title: "Configurar notificaciones"
description: "Canales, horarios, anti-fatiga."
---

`Ajustes → Notificaciones`.

## Por canal

Activa/desactiva cada canal globalmente:

| Canal | Plataforma |
|-------|-----------|
| Push móvil | iOS / Android |
| Push web | Navegador |
| Desktop toast | Tauri (Win/Mac/Linux) |
| In-app toast | Todas |
| Overlay (desktop) | Desktop |
| Email | Todas |

## Por tipo

Para cada tipo de notificación (Morning Briefing, Streak Alert, etc.):
- Activar / desactivar.
- Canal preferido.
- Horario aplicable.

## Quiet hours

Define una franja diaria sin notificaciones (ej: 22:00–07:00). Las notificaciones críticas (achievement, calendar change) pueden saltarla si lo permites.

## Límite diario

Tope total de notificaciones por día (default 5, max 20). Si excedes, la IA elige las más relevantes.

## Por hábito

Cada hábito puede tener su propio recordatorio:
- Hora fija.
- Smart (la IA elige el mejor momento).
- Off.

## Test

`Ajustes → Notificaciones → Enviar de prueba` para verificar que cada canal funciona en tu dispositivo.
