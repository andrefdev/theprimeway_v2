Las notas son tu segundo cerebro dentro de ThePrimeWay. Markdown nativo, vinculaciones a tareas, hábitos y objetivos.

## Crear una nota

`Notas → +` o atajo desde quick capture. Acepta markdown completo: headings, listas, code blocks, tablas, imágenes.

## Vincular

Cada nota puede asociarse a:
- Un objetivo (cualquier nivel).
- Una o varias tareas.
- Un hábito.

Al ver la nota, ves las entidades vinculadas. Al ver una tarea, ves sus notas.

## Tags y carpetas

- **Tags** libres para clasificación transversal.
- **Carpetas** opcionales para estructura jerárquica.

## Búsqueda

`Ctrl/Cmd + K` busca por título, contenido y tags.

## Adjuntos

Drag & drop de archivos (imágenes, PDFs) en una nota. Se almacenan asociados a tu cuenta.

## Templates

Templates pre-construidos: Daily Journal, Weekly Review, Reading Notes, Meeting Notes.
