La IA en ThePrimeWay no vive en una pestaña aparte. Está embebida en cada interacción donde puede aportar contexto: al crear una tarea, al perder un streak, al cambiar tu calendar, al revisar tu trimestre.

## Dónde se manifiesta

| Contexto | Acción de IA |
|----------|-------------|
| Abrir app por la mañana | Morning Briefing |
| Crear objetivo | Sugiere sub-objetivos coherentes |
| Crear tarea | Sugiere timebox + slot en calendar |
| Completar tarea | Muestra impacto en objetivo padre + siguiente sugerida |
| Inicio de trimestre | Quarterly Review automático |
| Streak en riesgo | Escalada de recordatorios |
| Cambio en Google Calendar | Re-agenda tareas afectadas |
| Domingo / Lunes | Weekly planning |
| Tarea abierta | Context brief + subtareas sugeridas |

## Privacidad y datos

- Solo se envían a la IA los datos necesarios para cada interacción (objetivos activos, tareas pendientes, eventos del día).
- No se entrenan modelos públicos con tus datos.
- Puedes desactivar la IA por completo en `Ajustes → IA → Activar`.
