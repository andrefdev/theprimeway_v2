## Anatomía de una tarea

- **Título** (obligatorio).
- **Descripción** — markdown soportado.
- **Prioridad** — low / medium / high.
- **Tags** — etiquetas libres.
- **Subtareas** — checklist con estado independiente.
- **Programación** — fecha + hora de inicio/fin estimadas.
- **Tiempo real** — start/end automáticos al usar Pomodoro o marcar inicio manual.
- **Vinculación** — a un objetivo de cualquier nivel.
- **Recurrencia** — diaria, semanal por día, mensual, custom.
- **Estado** — open / completed / archived.

## Estados y flujo

```
open ──► in progress ──► completed ──► (auto-archive en N días)
   └──► archived (manual)
```

## Backlog

Vista especial: tareas sin fecha asignada. Punto de partida del weekly planning. `Tareas → Backlog`.
