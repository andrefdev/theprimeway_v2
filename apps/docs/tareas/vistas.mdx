## Lista

Vista por defecto. Ordenable por: prioridad, fecha, estado, objetivo. Acciones rápidas: drag para reordenar, swipe (mobile) para completar.

## Kanban

Columnas: **Backlog · Hoy · En progreso · Completadas**. Drag & drop entre columnas. Muestra prioridad por color en el borde.

## Calendar

Tareas con `scheduledStart` aparecen en una vista semanal/mensual. Drag & drop para mover de día u hora. Integrado con eventos de Google Calendar (en gris, distinto color para distinguirlos).

## Timeline

Línea de tiempo horizontal. Útil para ver dependencias y carga de trabajo en las próximas 1-4 semanas.

## Filtros

Combinables en cualquier vista:
- Estado (open/completed/archived).
- Prioridad.
- Tag.
- Objetivo vinculado.
- Rango de fechas.
- Pomodoro asociado.

## Atajos

| Atajo | Acción |
|-------|--------|
| `1`, `2`, `3`, `4` | Cambiar entre vistas |
| `J` / `K` | Bajar / subir en la lista |
| `X` | Marcar tarea seleccionada como completada |
| `E` | Editar tarea seleccionada |
| `D` | Posponer (defer) |
