## Qué NO se hizo (deferred)

Refactors de scope amplio dejados como TODO. Mejor en commits dedicados:

### typedRoutes migration
- `app.json` ya tiene `typedRoutes: true` habilitado
- Código usa string literals: `href="/(auth)/login"`, `router.push('/(app)/pomodoro')`
- **Acción:** migrar a tipos generados en cada `<Link>` y `router.push()`. Decenas de callsites; ejecutar con `tsc --noEmit` para detectar typos.

### expo-image migration
- Package instalado, raramente usado
- `<Image>` de RN dispersa por el código
- **Acción:** sustituir todas las `<Image>` por `expo-image` con blurhash; mejora caché + carga progresiva.

### Suspense + skeletons en pantallas
- `Skeleton` y `SkeletonList` ya creados en `src/shared/components/feedback/Skeleton.tsx`
- Falta integrar en pantallas con loading states inconsistentes
- **Acción:** activar Suspense mode en react-query (`useSuspenseQuery`), envolver pantallas en `<Suspense fallback={<SkeletonList />}>`.

### Test manual en device
- TypeScript pass (`tsc --noEmit` clean)
- Falta probar golden path en simulador iOS/Android
- **Comando:** `cd apps/mobile && pnpm ios` (o `pnpm android`)

### Golden path manual
1. Login → home
2. QuickAction "Brain" → capture entry → ver auto-poll status hasta `complete`
3. QuickAction "Focus" (Pomodoro) → start/pause → background 30s → resume (verificar timer correcto)
4. Tasks tab → tap task → edit → save (toast)
5. Calendar tab → switch Month/Week/Day → tap evento → quick view → edit/delete
6. AI tab → mandar mensaje → ver streaming + markdown render + tool call card
7. Settings → enable biometric → background → resume → unlock o "Use password"
8. Modo avión → crear task → restaurar red → verificar sync (offline mutation queue)

## Cómo seguir la próxima sesión

1. **Probar en device primero** — `pnpm ios` y ejecutar golden path. Reportar bugs encontrados antes de seguir refactorizando.
2. **typedRoutes migration** — pasada completa, generar tipos con `pnpm expo router build` o similar, sustituir string literals.
3. **expo-image migration** — grep `from 'react-native'` con `Image` y migrar.
4. **Suspense integration** — empezar por una pantalla piloto (tasks today o brain feed) y replicar.
5. **Decidir tabs** — el 5to tab (`ai`) puede dejarse o reemplazarse por Brain si AI Chat usa menos. Consultar uso real.
6. **Optional follow-ups del audit que no estaban en el plan:**
   - Mutation ordering offline (riesgo de mutaciones paralelas que violan invariantes)
   - WebSocket/SSE para notifications (actualmente polling)
   - Sentinel/Sentry para crash reporting + sanitización stack
   - Limpiar fallback `created_at`/`started_at` en `ProactiveSuggestionsCard.tsx` (ya no necesario tras quitar transformer)
