## Widgets disponibles

| Widget | Tamaño | Qué muestra |
|--------|--------|-------------|
| **Tareas hoy** | Mediano / Grande | Próximas 3-5 tareas con tap para completar |
| **Streak counter** | Pequeño | Streak más largo activo |
| **Próximo hábito** | Pequeño / Mediano | Hábito pendiente más cercano + check-in tap |
| **Progreso del día** | Mediano | % tareas + hábitos completados |
| **Foco trimestral** | Grande | Foco activo + progreso + próxima acción |

## Quick check-in

Tocar la casilla del hábito en el widget lo marca como completo sin abrir la app. Animación de confirmación.

## Refresh

Los widgets se refrescan:
- Cada vez que abres y cierras la app.
- Cuando recibes una push notification.
- Cada ~15 minutos en background (depende del SO).

## Añadir widgets

<Tabs>
  <Tab title="iOS">
    Long press en pantalla inicio → **Editar pantalla** → **+** → Buscar "ThePrimeWay" → elige tamaño.
  </Tab>
  <Tab title="Android">
    Long press en pantalla inicio → **Widgets** → busca "ThePrimeWay" → arrastra a la pantalla.
  </Tab>
