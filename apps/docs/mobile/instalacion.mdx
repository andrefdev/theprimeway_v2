---
title: "Instalar Mobile"
description: "iOS App Store y Google Play."
---

<Tabs>
  <Tab title="iOS">
    1. Abre App Store y busca **ThePrimeWay**.
    2. Instala.
    3. Inicia sesión con tu cuenta o regístrate.
    4. Concede permisos: Notificaciones, Calendarios (opcional, para sync), Concentración / Focus.
    5. Añade widgets desde la pantalla de inicio: long press → Editar → +.
    
    Requisitos: iOS 16+.
  </Tab>
  <Tab title="Android">
    1. Abre Google Play y busca **ThePrimeWay**.
    2. Instala.
    3. Inicia sesión.
    4. Concede permisos: Notificaciones, Calendar (opcional), Permitir mostrar sobre otras apps (para overlay flotante en versiones avanzadas).
    5. Añade widgets desde el launcher.
    
    Requisitos: Android 9+.
  </Tab>
</Tabs>

## Sign-in con biometría

Tras el primer login, activa Face ID / Touch ID / huella en `Ajustes → Seguridad → Desbloqueo biométrico`.
