ThePrimeWay gamifica el progreso real, no la actividad superficial.

## Fuentes de XP

| Acción | XP base |
|--------|---------|
| Completar tarea | Variable según prioridad y duración |
| Mantener hábito (día) | Por hábito |
| Sesión Pomodoro completa | Por bloque |
| Completar reto diario | Bonus |
| Streak bonus | Por hábito con streak alto |
| Achievement desbloqueado | Bonus único |

## Ponderación por impacto

XP se multiplica según vinculación a objetivos:

| Vinculación | Multiplicador |
|------------|---------------|
| Sin objetivo | x1 |
| Vinculada a meta semanal | x1.5 |
| Vinculada a foco trimestral | x3 |
| Vinculada a objetivo anual | x2 |
| Vinculada a estratégico | x2.5 |

Esto premia trabajar en lo importante, no en lo urgente o fácil.

## Niveles y ranks

Tu XP total te coloca en un rango:

| Rank | Símbolo | Significado |
|------|---------|------------|
| **E** | Inicio | Estás empezando |
| **D** | Foundation | Construyendo bases |
| **C** | Consistent | Avance regular |
| **B** | Dedicated | Compromiso fuerte |
| **A** | Mastery | Resultados sostenidos |
| **S** | Sovereign | Élite, reservado para usuarios excepcionales |

Cada rank tiene niveles internos (D-1, D-2, ...). Subir de rank otorga celebración visual y achievement.

## Anti-fatiga

La IA detecta si estás haciendo solo tareas fáciles para acumular XP y propone retos reales. No es trampa, pero tampoco es progreso.
