## Cálculo automático del progreso

El % de cada objetivo se calcula en función de sus hijos:
- Tareas completadas vinculadas.
- Hábitos mantenidos según su frecuencia objetivo.
- Metas semanales logradas.
- Sub-objetivos cerrados.

No tienes que actualizarlo a mano. Lo que haces día a día mueve la aguja arriba.

## Indicador de salud

Cada objetivo muestra un semáforo calculado por la IA:

- 🟢 **Verde** — on track. Vas al ritmo necesario para el deadline.
- 🟡 **Amarillo** — en riesgo. Has bajado el paso.
- 🔴 **Rojo** — rezagado. Necesita atención o re-scope.

La IA considera: progreso actual, tiempo restante, ritmo histórico y dependencias.

## Vista árbol

`Objetivos → Vista árbol`: cascada visual desde tu visión hasta las tareas. Útil para chequear si toda tu actividad diaria está realmente alimentando tus prioridades.

## Snapshots semanales

Cada semana, ThePrimeWay guarda un snapshot del estado de cada objetivo. Esto te permite ver:
- **Tendencia**: ¿está mejorando, estancado o empeorando?
- **Momentum**: cuántas semanas consecutivas con progreso real.
- **Comparativas**: progreso semana actual vs media de las últimas 4.

`Objetivos → [tu objetivo] → Historial`.

## Alertas de inactividad

Si un objetivo no recibe actividad (tareas o hábitos) durante 14 días, la IA te notifica. Puede ser señal de:
- El objetivo dejó de ser relevante → archívalo.
- Bloqueo emocional → habla con la IA para desentrancar.
- Falta de descomposición → necesita más tareas concretas.
