## Email

- **Soporte general**: soporte@theprimeway.app
- **Privacidad**: privacy@theprimeway.app
- **Comercial / partnerships**: hello@theprimeway.app

Tiempo de respuesta: 24-48h en días laborables. Pro tiene prioridad.

## Status

Estado en tiempo real de la API y servicios: [status.theprimeway.app](https://status.theprimeway.app).

## Reportar un bug

1. `Ajustes → Ayuda → Reportar bug` desde la app (envía contexto técnico anonimizado).
2. O email a soporte@theprimeway.app con:
   - Plataforma (web/desktop/mobile + versión).
   - Pasos para reproducir.
   - Screenshot o video corto si aplica.

## Pedir una feature

- En app: `Ajustes → Ayuda → Sugerir feature`.
- Roadmap público: [theprimeway.app/roadmap](https://theprimeway.app/roadmap).
