## Características clave

- **Cierre = minimizar a tray** (estilo Discord). La app sigue corriendo: timers activos, notificaciones, overlay.
- **Overlay flotante** always-on-top, transparente, con tarea activa, timer y check-in rápido de hábitos.
- **Auto-start opcional** al iniciar el sistema.
- **Auto-updates silenciosos** en background.
- **Offline-capable** con cache SQLite local.
- **Window state persistence** (posición, tamaño).
- **Drag & drop** nativo de archivos a tareas/notas.
- **Deep links** `theprimeway://` para abrir secciones específicas.
