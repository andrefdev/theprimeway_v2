---
title: "Shortcuts globales"
description: "Atajos que funcionan aunque la app no tenga foco."
---

Estos shortcuts funcionan desde cualquier app. La app no necesita estar visible.

| Shortcut (Win/Linux) | Shortcut (macOS) | Acción |
|----------------------|------------------|--------|
| `Ctrl + Shift + O` | `Cmd + Shift + O` | Toggle overlay |
| `Ctrl + Shift + T` | `Cmd + Shift + T` | Iniciar / reanudar timer Pomodoro |
| `Ctrl + Shift + P` | `Cmd + Shift + P` | Pausar timer |
| `Ctrl + Shift + N` | `Cmd + Shift + N` | Quick capture de tarea |
| `Ctrl + Shift + D` | `Cmd + Shift + D` | Completar tarea activa |
| `Ctrl + Shift + M` | `Cmd + Shift + M` | Mostrar / restaurar dashboard |

## Atajos in-app (cuando la app tiene foco)

| Atajo | Acción |
|-------|--------|
| `Ctrl/Cmd + K` | Búsqueda global |
| `Ctrl/Cmd + I` | Abrir chat IA |
| `J / K` | Navegar en lista |
| `X` | Marcar item completado |
| `E` | Editar item seleccionado |
| `1`–`4` | Cambiar entre vistas (lista, kanban, calendar, timeline) |

## Personalizar

`Ajustes → Shortcuts`. Cambia cualquier combinación. Conflictos con shortcuts del sistema se detectan y avisan.

<Warning>
En macOS, los shortcuts globales requieren el permiso de **Input Monitoring** y **Accesibilidad**. La app te pedirá ambos en el primer arranque.
</Warning>
